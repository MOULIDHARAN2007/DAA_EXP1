# 🔍 Search Algorithm Visualizer

An interactive web-based visualization tool to understand and compare **Binary Search** and **Interpolation Search** through step-by-step animations, explanations, and performance analysis.

## 📖 Overview

This project helps students and beginners learn searching algorithms by visualizing every step of the search process. Instead of simply displaying the result, the application explains how each algorithm works, highlights the elements being visited, and compares their performance.

---

## ✨ Features

- 🎯 Step-by-step Binary Search visualization
- 🎯 Step-by-step Interpolation Search visualization
- ⚖️ Side-by-side algorithm comparison
- 📚 Detailed explanation for every step
- 🧮 Interpolation formula calculation
- 📊 Performance comparison
- 📈 Execution time analysis
- 🔄 Previous, Next, Reset, and Auto Play controls
- 🌙 Dark/Light mode

## 🚀 How It Works

### Binary Search

1. Initialize Low and High.
2. Calculate Mid.
3. Compare the middle element with the target.
4. Move either Low or High.
5. Repeat until the element is found or the search space becomes empty.

### Interpolation Search

1. Initialize Low and High.
2. Estimate the probable position using the interpolation formula.
3. Compare the estimated position with the target.
4. Update Low or High accordingly.
5. Continue until the element is found.

---

## 📊 Performance Comparison

| Algorithm | Average Time Complexity | Worst Time Complexity | Space Complexity |
|------------|------------------------|-----------------------|------------------|
| Binary Search | O(log n) | O(log n) | O(1) |
| Interpolation Search | O(log log n) | O(n) | O(1) |

## 🎮 How to Use

1. Clone the repository.

```bash
git clone https://github.com/yourusername/Search-Algorithm-Visualizer.git
```

2. Open the project folder.

3. Double-click `index.html`

or

Open it using VS Code Live Server.

4. Enter a sorted array.

5. Enter the target value.

6. Click **Start Visualization**.

7. Watch the algorithms execute step by step.
